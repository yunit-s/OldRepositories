Response Headers 확인하는 방법

브라우저에서 f12 - Network 탭 열어둔 채로 웹페이지 새로고침 해보면 왼쪽에 뭐 뜬다. 그거 클릭하면 된다.

Test Link 1

text/plain 호출(일반 문자열 리턴)
브라우저에서 결과물 확인
Status Code: 200 (성공 코드)
Content-Type: text/plain;charset=UTF-8

Test Link 2

text/plain 호출(ResponseEntity 리턴)
브라우저에서 결과물 확인
Status Code: (CREATED를 리턴했을 때) 201
Content-Type: text/plain;charset=UTF-8 (headers.setContextType 에 적은 것을 따라간다.)

Test Link 3

application/json 호출(VO 리턴)
브라우저에서 결과물 확인
Status Code: 200
Content-Type: application/json;charset=UTF-8
JSON Formatter 확장 프로그램을 이용하여, JSON 데이터를 화면에 띄워준다.
